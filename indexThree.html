why do we need function

Help us reduce code redundancy by allowing us to write a function once and call it multiple times.
Make code modular by allowing us to break a big file into smaller, more manageable chunks.
Provide abstraction by allowing us to use library functions without worrying about their internal work.
Allow us to analyse, design, investigate, predict, or solve various events.
Serve as a tool to better understand the world, create solutions, improve products and services, or make logical choices.
